## Overview
A high-performance, cloud-based development environment built using a Next.js TypeScript frontend and a PHP REST API. This platform provides a full-featured code editor with file system persistence and workspace management.

## Features
- **PHP 8.x**: Handles core business logic, file system operations, and MySQL database interactions.
- **Next.js 15**: Provides a reactive, server-side rendered interface for the code editor and dashboard.
- **Monaco Editor**: Integrates the same core editor powering VS Code for high-quality syntax highlighting and IntelliSense.
- **MySQL**: Manages user accounts, project metadata, and session persistence.
- **Xterm.js**: Implements a browser-based terminal for command-line simulation.
- **GitHub OAuth**: Secure authentication flow for developer accounts.

## Getting Started
### Installation
1. **Clone the Repository**
   ```bash
   git clone https://github.com/apreezofficial/ap-web-ide.git
   cd ap-web-ide
   ```

2. **Database Setup**
   - Open XAMPP/WAMP and start MySQL.
   - Import the `server/database.sql` file into your database manager.
   - Create a database named `ap_ide`.

3. **Backend Configuration**
   - Navigate to `server/config/config.php`.
   - Update your database credentials (`DB_USER`, `DB_PASS`).
   - Fill in your `GITHUB_CLIENT_ID` and `GITHUB_CLIENT_SECRET`.

4. **Frontend Setup**
   - Open a terminal in the `client` directory.
   ```bash
   npm install
   npm run dev
   ```

### Environment Variables
**Client (.env.local)**
```text
NEXT_PUBLIC_APP_ENV=local
```

**Server (config.php constants)**
```php
define('DB_HOST', 'localhost');
define('DB_NAME', 'ap_ide');
define('DB_USER', 'root');
define('DB_PASS', '');
define('GITHUB_CLIENT_ID', 'your_id');
define('GITHUB_CLIENT_SECRET', 'your_secret');
```

## API Documentation
### Base URL
`http://localhost/ap-ai-ide/server/api`

### Endpoints

#### GET /auth/user.php
**Request**:
No payload required. Requires session cookie.

**Response**:
```json
{
  "authenticated": true,
  "user": {
    "id": 1,
    "username": "johndoe",
    "email": "john@example.com",
    "avatar_url": "https://github.com/..."
  }
}
```

**Errors**:
- 200: `{"authenticated": false}` if no session exists.

#### POST /projects/create.php
**Request**:
```json
{
  "name": "my-new-app",
  "template": "blank"
}
```

**Response**:
```json
{
  "success": true,
  "id": 12
}
```

**Errors**:
- 401: Unauthorized
- 400: Project name is required
- 500: Failed to create workspace directory

#### GET /projects/list.php
**Request**:
No payload required.

**Response**:
```json
{
  "projects": [
    {
      "id": 1,
      "name": "portfolio-site",
      "path": "workspaces/1/portfolio-site",
      "last_accessed": "2023-10-27 10:00:00"
    }
  ]
}
```

**Errors**:
- 401: Unauthorized

#### GET /files/list.php
**Request**:
Query Parameters: `project_id` (Required), `path` (Optional)

**Response**:
```json
{
  "files": [
    {
      "name": "index.php",
      "path": "index.php",
      "type": "file"
    },
    {
      "name": "assets",
      "path": "assets",
      "type": "directory"
    }
  ]
}
```

**Errors**:
- 400: Project ID required
- 404: Project not found
- 401: Unauthorized

#### POST /files/write.php
**Request**:
```json
{
  "project_id": 1,
  "path": "index.php",
  "content": "<?php echo 'hello world'; ?>"
}
```

**Response**:
```json
{
  "success": true
}
```

**Errors**:
- 401: Unauthorized
- 400: Project ID and Path required
- 500: Disk write permission error

#### GET /files/read.php
**Request**:
Query Parameters: `project_id`, `path`

**Response**:
```json
{
  "content": "File content string here..."
}
```

**Errors**:
- 404: File or Project not found
- 401: Unauthorized
